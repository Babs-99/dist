HTML Cheat Sheet

Tag Description Example
<html> Defines the root of an HTML document 
<html></html>
<head> Contains metadata about the HTML document 
<head></head>
<body> Sets off the visible portion of the document 
<body></body>
<title> Puts name of the document in the title bar 
<title></title>
<div> Used to format block content with CSS 
<div></div>
<ul> Creates an unordered list 
<ul></ul>
<li> Encompasses each list item 
<li></li>
<span> Used to format inline content with CSS 
<span></span>
<h1> -- <h6> Creates headlines. H1=largest, H6=smallest 
<h1></h1>
<hr> Inserts a horizontal rule 
<hr>
<br> Inserts a line break 
<br>
<blockquote> Puts content in a quote 
<blockquote></blockquote>
<form> Defines a form 
<form></form>
<table> Creates a table 
<table></table>
<tr> Sets off each row in a table 
<tr></tr>
<td> Sets off each cell in a table 
<td></td>
<th> Sets off the table header 
<th></th>
